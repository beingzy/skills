# OpenCode Agent

Use this skill as a compact design-review and implementation overlay for flat, quiet, minimal frontend work.

## Invocation

```text
Use $flat-minimal to make this interface quieter, flatter, and more self-contained.
```

## Apply When

- polishing brand kits, docs, landing pages, and public product pages
- tuning layout, spacing, card density, or visual hierarchy
- converting bordered card grids into softer structure
- checking whether public-facing content feels external-ready

## Implementation Hints

- Keep prose headers in a readable narrow column.
- Let important groups break wider.
- Use pale, borderless enclaves when spacing can carry containment.
- Increase card width before shrinking type.
- Increase surface contrast slightly before adding a border.
- Verify mobile width and rendered asset fidelity before finishing.
