# Claude Code Agent

Use this skill as a visual preference overlay when Claude Code is editing frontend files, brand pages, docs pages, landing pages, or public product pages.

## Invocation

```text
Use $flat-minimal with the project design context to polish this page.
```

## Operating Notes

- Load `SKILL.md` first.
- Load `references/patterns.md` when implementing CSS, layout, or spacing changes.
- Pair with existing project instructions, brand guidelines, screenshots, and visual inspection.
- Treat the skill as taste direction, not a replacement for product requirements.

## Review Bias

Prefer:

- outline-less enclaves
- selective width breakouts
- calm spacing
- flat pale surfaces
- factual asset fidelity
- external-facing clarity

Avoid:

- visible structure added too early
- generic SaaS card grids
- borders used to compensate for weak spacing
- internal product examples on public brand pages
- mobile layouts that merely collapse without composition
